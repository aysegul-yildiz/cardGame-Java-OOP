# cardGame-Java-OOP
This is a java based implementation of a four player card game, developed as part of CS 102 course assignment in 2022.
The project focuses on  Object-Oriented Programming (OOP) principles including encapsulation, modularity, and package management.
The game manages a full deck of 52 cards distributes them among four players and tracks scores across multiple rounds based on the card values played.
Card: Represents a single playing card with face value and suit
Cards: A collection manager for card sets used for both the main deck and player hands.
Player: Stores player data including their name and current hand.
CardGame: The central controller that manages game state, turns and winner.
ScoreCard: An analytics class that tracks and displays player scores throughout the session.
